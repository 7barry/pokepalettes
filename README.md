# PokePalettes

16-bit palettes that represent Pok√©mon. [PokePalettes](https://opensea.io/collection/pokepalettes/) is an NFT collection on OpenSea. Feel free to use these palettes for anything. If you do, [start a new issue](https://github.com/BarryMode/pokepalettes/issues/new) to share how you used it. Please include links or pictures so everyone who comes here will have a showcase.

## Build

#### html

```bash
python palettes_to_html.py
```

#### png

```bash
./palettes_to_png.sh
```

#### txt

```bash
python palettes_to_txt.py
```
